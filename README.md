# TXRJets Media Assets
Digital assets and audio files for the TXRJets ecosystem.

## Usage
To use these files in the web player or dashboard:
1. Click the file name.
2. Click **"Raw"**.
3. Use the URL in your `<audio>` or `<video>` tags.

## Optimization
Files are hosted here for low-latency delivery across the TXRJets General Dashboard and marketing assets.
